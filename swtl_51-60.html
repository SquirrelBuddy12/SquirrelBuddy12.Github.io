Functions: Local vs. Global Variables

Now we come to the subject of variable scope. That is, the difference between global and local variables. Some variables have global scope, which makes them global variables. Other variables have local scope, which makes them local variables. A global variable is one that's declared in the main body of your codeâ€”that is, not inside a function. A local variable is one that's declared inside a function. It can be either a parameter of the function, which is declared implicitly by being named as a parameter, or a variable declared explicitly in the function with the var keyword.
